Detailed Description

Структура, представляющая таблицу строк (String Table) в формате BIFF (Binary Interchange File Format) для Excel.

Таблица строк используется в Excel для хранения всех строковых данных (например, текстовые метки, надписи в ячейках, и т.д.), которые могут быть повторно использованы в различных частях документа.
